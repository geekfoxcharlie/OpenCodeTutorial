# OpenCode 中文指南 (OpenCode Chinese Guide)

> **零基础普通人的 AI 编程智能体入门手册**

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

OpenCodeTutorial 是一个致力于普及 AI 编程智能体（Agent）的开源项目。我们用最通俗易懂的语言，手把手教你如何使用 OpenCode 重塑你的编程体验。

## 🌟 项目特色

- **零基础友好**：不讲晦涩的代码术语，用“超级实习生”等比喻带你入门。
- **中国特供优化**：针对国内网络环境，提供 DeepSeek、通义千问等国产模型的最优接入方案。
- **实战导向**：包含从安装、配置到跨文件重构、Bug 排查的全流程实战工作流。
- **纯净体验**：基于静态 HTML + Tailwind CSS 构建，响应式设计，极速加载。

## 📚 课程大纲

1. **什么是 OpenCode？** —— 建立对 AI 智能体的正确认知。
2. **安装指南** —— 全平台一键安装及 Zen 模型接入。
3. **国内特供版** —— 国产模型（DeepSeek 等）实战配置。
4. **从 Cursor 迁移** —— 老手的快速平滑过渡指南。
5. **详细配置** —— 定制你的专属 AI 别名与 Provider。
6. **实战工作流** —— 像专家一样提问（重构、测试、Debug）。
7. **常用技巧** —— 掌控 Plan/Build 模式与上下文引用。
8. **高手进阶** —— 探索 MCP 协议与 UltraWork 模式。

## 🚀 本地运行

本项目不需要复杂的构建环境，只需一个简单的静态服务器即可。

### 方式一：使用 Python (推荐)

```bash
# 启动开发服务器 (默认端口 8000)
npm run dev
```

### 方式二：使用 Node.js (如果你有安装)

```bash
npm install
npm run preview
```

访问 `http://localhost:8000/guide/index.html` 即可预览。

## 🛠 技术栈

- **样式**: Tailwind CSS (CDN)

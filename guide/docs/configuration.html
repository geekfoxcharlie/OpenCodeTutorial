第五章：配置详解 —— 定制你的专属 AI

OpenCode 的强大在于其极高的可配置性。配置文件通常位于 ~/.config/opencode/config.json (Linux/macOS) 或 %APPDATA%/opencode/config.json (Windows)。 

{
  "providers": {
    "opencode": { "id": "opencode" },
    "openai": {
      "id": "openai",
      "args": { "apiKey": "sk-..." }
    }
  },
  "models": {
    "default": "opencode/default"
  }
}

1. 接入 DeepSeek Coder V3 (高性价比)

DeepSeek 是目前最适合编程的高性价比模型。你可以通过 OpenRouter 或 DeepSeek 官方 API 接入。 

{
  "providers": {
    "deepseek": {
      "id": "openai",
      "args": {
        "baseUrl": "https://api.deepseek.com/v1",
        "apiKey": "sk-your-deepseek-key",
        "models": {
          "coder": { "id": "deepseek-coder" },
          "chat": { "id": "deepseek-chat" }
        }
      }
    }
  },
  "models": {
    "default": "deepseek/coder"
  }
}

2. 接入本地 Ollama (隐私优先)

如果你拥有强劲的本地显卡（如 RTX 4090），可以使用 Ollama 运行 Llama 3 或 DeepSeek 本地版，实现完全免费且隐私的编程助手。 

{
  "providers": {
    "ollama": {
      "id": "openai",
      "args": {
        "baseUrl": "http://localhost:11434/v1",
        "apiKey": "ollama",
        "models": {
          "llama3": { "id": "llama3:70b" },
          "deepseek": { "id": "deepseek-coder:33b" }
        }
      }
    }
  }
}

3. 设置别名 (Aliases)

不想每次都打 anthropic/claude-3-5-sonnet-20240620 这么长？设置别名吧。 

{
  "models": {
    "sonnet": "anthropic/claude-3-5-sonnet-20240620",
    "gpt4o": "openai/gpt-4o",
    "ds": "deepseek/coder"
  }
}

设置后，你可以在启动时直接使用：opencode --model sonnet。

💡 提示

修改配置文件后，无需重启终端，但在 OpenCode 交互界面中可能需要重启会话才能生效。

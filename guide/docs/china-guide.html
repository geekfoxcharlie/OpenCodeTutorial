第三章：中国大陆使用指南

🚀 针对国内网络环境、支付方式和模型选择优化的专项指南。

💡 核心优势

国内用户使用 OpenCode 的核心优势在于：你可以随意切换到国产模型（DeepSeek, 通义千问, 智谱）。这不仅意味着更低的成本，更意味着极致的访问速度和稳定性。

1. 网络问题排查

如果你在安装或更新时遇到网络超时或证书错误，试试以下方案：

安装时的网络问题

# 方案 A：使用国内镜像源安装 Node.js (如果你用 npm 安装)
npm install -g opencode-ai --registry=https://registry.npmmirror.com

# 方案 B：使用 Docker (如果网络连接受限)
docker run -it --rm ghcr.io/anomalyco/opencode

API 调用的网络问题

如果调用 OpenAI/Anthropic 官方 API 失败，你有两个选择：

  • 方案一：使用国内中转 API（推荐）
    购买 OpenAI/Claude 的中转 Key，价格通常比官方贵 20-50%，但稳定性好。
  • 方案二：切换到国产大模型（性价比最高）
    直接使用 DeepSeek 或通义千问，不仅免费，而且速度快。

2. 国产模型配置实战

DeepSeek (目前最推荐)

DeepSeek Coder 在编程任务上的表现已经非常接近 Claude 3.5，且 API 极其便宜。 

{
  "providers": {
    "deepseek": {
      "id": "openai",
      "args": {
        "baseUrl": "https://api.deepseek.com/v1",
        "apiKey": "sk-your-api-key",
        "models": {
          "default": { "id": "deepseek-coder" }
        }
      }
    }
  },
  "models": {
    "default": "deepseek/default"
  }
}

通义千问 (阿里云)

如果你有阿里云账号，可以使用 DashScope 模型。

{
  "providers": {
    "aliyun": {
      "id": "openai",
      "args": {
        "baseUrl": "https://dashscope.aliyuncs.com/compatible-mode/v1",
        "apiKey": "sk-your-dashscope-key",
        "models": {
          "qwen-turbo": { "id": "qwen-turbo" },
          "qwen-plus": { "id": "qwen-plus" }
        }
      }
    }
  }
}

智谱 AI (ChatGLM)

{
  "providers": {
    "zhipu": {
      "id": "openai",
      "args": {
        "baseUrl": "https://open.bigmodel.cn/api/paas/v4",
        "apiKey": "your-zhipu-api-key",
        "models": {
          "glm-4": { "id": "glm-4" }
        }
      }
    }
  }
}

3. 支付与账户问题

⚠️ 关于 OpenCode Zen

OpenCode Zen (官方提供的模型服务) 目前对国内用户最不友好，因为它需要海外信用卡。如果你没有外币卡，请务必参考上面的“国产模型配置”，直接配置第三方 API。

4. 推荐组合 (2025-2026)

经过社区实测，以下组合在国内体验最好：

💰 省钱组合

  • 主力：DeepSeek Coder
  • 兜底：通义千问 Qwen-Turbo
  • 月成本：≈ ¥50

⚡ 速度组合

  • 主力：通义千问 Qwen-Plus
  • 特点：延迟极低
  • 适用：日常小改动

🧠 质量组合

  • 主力：Claude 3.5 (需中转)
  • 适用：复杂重构
  • 成本：较高

第二章：快速上手 —— 安装与配置

1. 安装 OpenCode

OpenCode 支持全平台安装。最简单的方法是使用安装脚本：

curl -fsSL https://opencode.ai/install | bash

针对不同平台的补充：

  • macOS (Homebrew): brew install anomalyco/tap/opencode
  • Windows: choco install opencode 或 scoop install extras/opencode
  • Docker: docker run -it --rm ghcr.io/anomalyco/opencode

2. 配置 LLM 服务商

初次启动建议使用 OpenCode Zen，这是官方提供的模型中转服务，经过深度优化。

  1. 在终端输入 opencode 启动。
  2. 执行命令 /connect。
  3. 在弹出的界面中选择 opencode，跳转到浏览器登录并获取 API Key。
  4. 回到终端粘贴 Key 即可完成。

如果你想使用自己的 API（如 Claude 3.5 Sonnet 或 DeepSeek），可以编辑配置文件 ~/.config/opencode/config.json。

3. 项目初始化

进入你的项目目录，运行初始化命令：

cd /your/project/path
opencode
/init

该命令会让 OpenCode 扫描你的项目结构，并生成 AGENTS.md。这对于它理解你的代码至关重要，请务必将此文件加入 Git 版本控制。
